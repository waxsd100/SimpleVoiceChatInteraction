# Simple Voice Chat Interaction

このサーバーサイドForge用Modは、Simple Voice Chatの音声をMinecraftの世界と相互作用させることができます。

## 機能

- ボイスチャットで喋るとスカルクセンサーが反応する
- ボイスチャットで喋るとウォーデンに検知される
- ボイスチャットで大声を出すと**ソニックショックウェーブ**が発動する（ディープダークバイオームおよび `deeperdarker:otherside` ディメンション限定）
  - **フェーズ1 — 周囲AoE**: プレイヤー周囲の小範囲（基準半径×0.5）にいる全エンティティにダメージ
  - **フェーズ2 — ソニックビーム**: プレイヤーの視線方向にウォーデン風の指向性ビームを発射（射程：基準半径×3、ダメージ：AoEの1.5倍）
  - ビーム被弾時のノックバックはダメージに比例してスケール
  - ビーム被弾プレイヤーにはスタン効果: 移動速度低下 Lv.200（移動不能）、盲目（視界が霧に覆われる）、ジャンプ封じ、暗闇
  - スタン持続時間は設定値に準拠
- **ショックウェーブ経路全体でスカルク振動が発生**
  - プレイヤー位置、ビーム沿い（8ブロック間隔）、着弾地点でスカルクセンサー/シュリーカーが反応
  - シュリーカーが密集しているエリアではウォーデン召喚の連鎖が発生しうる
- 声の大きさ（dB）に応じてショックウェーブの範囲とダメージが動的に変化する（最大200dB）
  - 100dBを超えると**オーバードライブ**状態となり、さらに強力な倍率が適用される
- **音声正規化（AGC）**: プレイヤー間のマイク感度差を自動補正。SVCの設定に関わらず、全員が同じ程度の音量でスカルク/ショックウェーブを発動する
- **ウール防音**: 周囲をウールブロックで囲むと音声レベルが減衰する。バニラのスカルクセンサーもウールで振動が遮断される仕様と一貫。6方向すべてで最大-20dB
- 高度なノイズ抑制機能を搭載（バンドパスフィルターとゼロ交差率判定により、環境音や打鍵音を自動カット）
- モンスターおよびウォーデンに対するボーナスダメージ倍率
- グループチャットでの振動発生のオプション対応
- 囁き声（Whisper）時・スニーク時は声の大きさが自動的に小さくなる機能
- ダッシュ中は声の大きさが自動的に大きくなる機能
- ディープダーク（古代都市など）または `deeperdarker:otherside` ディメンションに入ると自動で表示される音量メーター（BossBar）
  - 音量に応じた色変化、発動時の特殊演出、クールダウンタイマー機能付き

## コマンド

- `/voice_meter`: 音量メーター（BossBar）の表示/非表示を手動で切り替えます
- `/voice_reload`: 設定ファイルをディスクから再読み込みします（要OP権限レベル2）
- `/voice_debug <dB>`: マイクを使わずに指定dBの音声入力をテストシミュレートします（要OP権限レベル2）
- `/voice_monitor [player]`: 指定した他プレイヤーの現在の音声レベル（dB）をBossBarにリアルタイム表示します。引数なしで実行するとモニタリングを停止します。

### ショックウェーブ & スカルク反応 早見表（デフォルト設定）

各音量レベルでの動作を示します:

| 音量 | スカルク反応 | ショックウェーブ | AoE半径 | ビーム射程 | 備考 |
|------|:-----------:|:---------------:|:-------:|:---------:|------|
| **60 dB**（普通の声） | ✅ | ❌ | — | — | スカルクセンサーが反応。ショックウェーブなし。 |
| **85 dB**（しきい値） | ✅ | ✅ | **2.0 m** | **12.0 m** | 最小ショックウェーブ。プレイヤー位置+ビーム沿い+着弾点でスカルク反応。 |
| **100 dB**（大声） | ✅ | ✅ | **5.0 m** | **30.0 m** | オーバードライブ前の最大。 |
| **200 dB**（オーバードライブ最大） | ✅ | ✅ | **10.0 m** | **60.0 m** | オーバードライブ×2.0適用。超広範囲。 |

> **ショックウェーブ時のスカルク振動ポイント:**
> - プレイヤー位置（AoE中心）
> - ビーム沿い8ブロックごと（スカルクセンサーの検知範囲に合致）
> - ビーム着弾地点（末端）
>
> 100dBのビームでは約4箇所の振動が発生。200dBでは最大約12箇所 — 複数のシュリーカーを連鎖起動しウォーデンを召喚するのに十分。

### ショックウェーブのスケーリング式

声の大きさ（dB）に応じて、ショックウェーブの威力と範囲が動的にスケールします。設定した閾値（`85dB`）では最小となり、最大音量（`100dB`）に向かって線形に増加します。さらに100dBを超えると、オーバードライブ係数に従って最大200dBまで天井知らずで強化されます。

**周囲AoE範囲**（基本半径×0.5）
- 最小音量時（閾値付近）: 半径 **2.0** ブロック
- 100dBの叫び声: 半径 **5.0** ブロック

**ソニックビーム射程**（基本半径×3）
- 最小音量時（閾値付近）: **12.0** ブロック
- 100dBの叫び声: **30.0** ブロック

**オーバードライブ係数**（100dB〜200dB、デフォルト `overdrive_multiplier = 2.0`）

100dB時の半径・ダメージにこの係数が乗算されます:

| 音量 | オーバードライブ係数 | 基本半径 | AoE半径 | ビーム射程 |
|:---:|:---:|:---:|:---:|:---:|
| **100 dB** | ×1.0 | 10.0 | **5.0 m** | **30.0 m** |
| **120 dB** | ×1.2 | 12.0 | **6.0 m** | **36.0 m** |
| **150 dB** | ×1.5 | 15.0 | **7.5 m** | **45.0 m** |
| **200 dB** | ×2.0 | 20.0 | **10.0 m** | **60.0 m** |

**周囲AoEダメージ**（デフォルト設定時）

| 対象 | 倍率 | 85 dB（しきい値） | 100 dB | 200 dB（オーバードライブ） |
|------|:---:|:---:|:---:|:---:|
| **プレイヤー** | `0.5x` | **2.0** (❤️×1) | **4.0** (❤️×2) | **8.0** (❤️×4) |
| **中立（動物等）** | `1.0x` | **4.0** (❤️×2) | **8.0** (❤️×4) | **16.0** (❤️×8) |
| **モンスター** | `10.0x` | **40.0** (❤️×20) | **80.0** (❤️×40) | **160.0** (❤️×80) |
| **ウォーデン** | `20.0x` | **80.0** (❤️×40) | **160.0** (❤️×80) | **320.0** (❤️×160) |

### 音量（dB SPL）の目安

#### ゲーム内スケール（マイク入力）

| dB | 説明 |
|:---:|---|
| **200 dB** | システムの最大上限。オーバードライブの天井。 |
| **100 dB** | マイクの限界値（音が割れる）。マイクに口を近づけて大声で叫んだり、マイクを叩いた時の音。 |
| **80 〜 90 dB** | かなり大きな声。ゲームで驚いて叫んだり、大爆笑している時の音量。 |
| **60 〜 70 dB** | 普通の会話レベル。Discordなどでリラックスして喋っている時の音量。 |
| **40 〜 50 dB** | 小さな音・雑音。囁き声、キーボードの打鍵音、マウスのクリック音、環境ノイズ。 |

#### 現実世界のdBリファレンス

> デシベル（dB）は**対数スケール**です。**+10 dB** ごとに人間の耳には約**2倍**の大きさに聞こえます。

| dB | 現実の音の例 |
|:---:|---|
| **194 dB** | **音の理論上の限界（大気圧下）。** 音は空気の圧力変化（波）です。1気圧の環境では、音波の最も低い部分（谷）が真空に達するため、これ以上の振幅は波として圧縮できません。これを超えると**衝撃波**となります。 |
| **200 dB** | 音響の限界を超える領域。圧力ブラスト（至近距離での核爆発など）でのみ実現可能。 |
| **160 〜 180 dB** | 至近距離での雷鳴。大規模な爆発。1883年のクラカタウ火山噴火は160km離れた場所で約180 dBを記録。 |
| **140 〜 150 dB** | ジェットエンジンの真横。**回復不能な聴覚損傷** — 鼓膜の破裂、内出血、持続的な曝露で肺の損傷。 |
| **130 dB** | 人間の苦痛の限界。身体的な不快感が耐えられないレベル。 |
| **120 dB** | 聴覚が痛みを感じ始める閾値。300m先の航空機離陸。長時間の曝露で永続的な聴力低下。 |
| **100 〜 110 dB** | ロックコンサートの最前列。チェーンソー。長時間の曝露には聴覚保護が必要。 |
| **80 〜 90 dB** | 交通量の多い道路。掃除機。1m先での怒鳴り声。 |
| **60 〜 70 dB** | 1m先での普通の会話。BGM程度の音楽。 |
| **40 〜 50 dB** | 静かなオフィス。囁き声での会話。 |
| **20 〜 30 dB** | 木の葉のざわめき。夜の静かな寝室。 |
