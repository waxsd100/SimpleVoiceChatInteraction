# Simple Voice Chat Interaction

このサーバーサイドForge用Modは、Simple Voice Chatの音声をMinecraftの世界と相互作用させることができます。

## 機能

- ボイスチャットで喋るとスカルクセンサーが反応する
- ボイスチャットで喋るとウォーデンに検知される
- ボイスチャットで大声を出すと、周囲のエンティティにダメージを与えるソニックショックウェーブが発生する（ディープダークバイオームおよび `deeperdarker:otherside` ディメンション限定）
- 声の大きさ（dB）に応じてショックウェーブの範囲とダメージが動的に変化する
- モンスターおよびウォーデンに対するボーナスダメージ倍率
- グループチャットでの振動発生のオプション対応
- 囁き声（Whisper）時は声の大きさが自動的に小さくなる機能（減少倍率を設定可能）
- スニーク中は声の大きさが自動的に小さくなる機能（減少倍率を設定可能）
- ダッシュ中は声の大きさが自動的に大きくなる機能（増幅倍率を設定可能）
- ディープダーク（古代都市など）または `deeperdarker:otherside` ディメンションに入ると自動で表示される音量メーター（BossBar）

## コマンド

- `/voice_meter`
  音量メーター（BossBar）の表示/非表示を切り替えます。
- `/voice_reload`
  設定ファイル（Config）の変更をディスクから再読み込みします。（要OP権限）
- `/voice_debug <dB>`
  マイクを使わずに、指定した dB の音声入力をシミュレートします。（要OP権限）

### ショックウェーブの攻撃範囲とダメージ表

声の大きさ（dB）に応じて、ショックウェーブの威力と範囲が動的にスケールします。最小音量（閾値：例 `80dB`）で倍率1.0倍となり、最大音量（`100dB`）に近づくにつれて設定した最大倍率まで増加します。

**攻撃範囲（半径）**
- 最小音量時: 半径 **10.0** ブロック
- 最大音量時: 半径 **20.0** ブロック（直径40ブロック）

**ダメージ表**（デフォルト設定時）

| 対象エンティティ | 基本倍率 | 最小ダメージ<br>(閾値ギリギリ) | 最大ダメージ<br>(100dBの叫び声) | 備考 |
|---|---|---|---|---|
| **プレイヤー** | `0.5倍` | **2.0** (❤️x1) | **4.0** (❤️x2) | 味方を巻き込んでも致命傷になりにくい |
| **動物・村人など** | `1.0倍` | **4.0** (❤️x2) | **8.0** (❤️x4) | 特効の乗らない標準ダメージ |
| **モンスター** | `5.0倍` | **20.0** (❤️x10) | **40.0** (❤️x20) | 普通の敵なら軽く声を出せば確殺 |
| **ウォーデン** | `20.0倍` | **80.0** (❤️x40) | **160.0** (❤️x80) | 全力で叫べば体力の1/3を削り取る超火力 |

### 音量（dB SPL）の目安

- **100 dB**：マイクの限界値（音が割れる）。マイクに口を近づけて大声で叫んだり、マイクを叩いたりした時の音。
- **80 〜 90 dB**：かなり大きな声。ゲームで驚いて「うわっ！」と叫んだり、大爆笑している時の音量。
- **60 〜 70 dB**：普通の会話レベル。Discordなどでリラックスして喋っている時の音量。
- **40 〜 50 dB**：小さな音・雑音。ささやき声や、マイクが拾ってしまう「キーボードのタイピング音」「マウスのクリック音」「遠くの車の音」など。
- **30 dB 以下**：ほぼ無音（マイク自体のサーッというホワイトノイズなど）。
